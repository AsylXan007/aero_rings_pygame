# aero_rings_pygame
Sort of a game made by me as a project for programming lessons
Game files should be contained in the Aero_Rings directory for them to work properly.

Controls:
w - move the plane upwards
a - move the plane to the left
s - move the plane downwards
d - move the plane to the right

b - turn on the bot for autoplay, disables other controls, pause and music off/on buttons still work
